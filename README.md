# Netset

Netset is a Django web application that connects job seekers with businesses.

## Features

- User Registration
- User Login & Logout
- Custom User Model
- User Profiles
- Job Seeker Accounts
- Business Owner Accounts
- Dashboard
- PostgreSQL Database

## Built With

- Python
- Django
- PostgreSQL
- HTML
- CSS

## Future Features

- Job Listings
- Job Applications
- Search
- Messaging
- AI Job Matching

## Installation

```bash
git clone https://github.com/egwaborehis/Netset.git

cd Netset

python -m venv .venv

.venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
